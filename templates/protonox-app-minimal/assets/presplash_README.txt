# Presplash Image Placeholder

This file should be replaced with your app presplash image.

Requirements:
- Format: PNG
- Size: 512x512 pixels (recommended)
- Will be shown while the app loads
- Keep it simple and branded

The presplash appears immediately when the app launches on Android
and disappears once Kivy has fully loaded.

Place your presplash.png file in this assets/ directory.
