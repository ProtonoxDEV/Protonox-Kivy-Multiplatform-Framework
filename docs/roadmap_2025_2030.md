# Roadmap 2025-2030

## 2025
- Ship Protonox devtools with live component inspector and remote console UI.
- Expand payment adapters (Nequi, Binance Pay, Stripe test harness).
- Add offline-first caching layer for Firestore with local persistence.

## 2026
- Introduce modular plugin system for native extensions (notifications, biometrics).
- Provide official Docker images for reproducible Buildozer environments.

## 2027
- macOS notarization pipeline examples and hardened runtime presets.
- Advanced analytics dashboard template with charting presets.

## 2028
- Multi-tenant theming engine with design tokens and dynamic color schemes.
- Built-in A/B testing hooks for UI experiments.

## 2029
- GPU-accelerated text/emoji rendering upgrades.
- Accessibility-first widgets and screen reader guidance.

## 2030
- AI-assisted UI composer for Kivy KV layouts.
- Full offline-capable backend sync adapters for Render/Firebase hybrids.
