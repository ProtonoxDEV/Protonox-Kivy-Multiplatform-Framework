# Protonox Agents

Este repositorio incluye agentes pensados para usarse con cualquier chatbot de VS Code (Copilot, Continue, Cursor, Codeium, ChatGPT Desktop, etc.). Copia los prompts desde `prompts/` y sigue el onboarding en `docs/mentor/START_HERE.md`.

- **Protonox Mentor (Positrón)**: guía interactiva para principiantes en Protonox Studio / Kivy Protonox. Ayuda a instalar, correr demos, crear pantallas, gestionar assets y solucionar errores comunes. Usa `protonox mentor start` o abre `docs/mentor/START_HERE.md`.
- **Mentor Tasks**: misiones cortas con checklist y validación manual. Ver `prompts/mentor.tasks.md`.

Uso rápido:
1) Abre `docs/mentor/START_HERE.md`.
2) Copia el prompt base desde `prompts/mentor.system.md` en tu chatbot.
3) Ejecuta `protonox mentor start` para abrir el tutorial y seguir las misiones.
