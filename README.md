# Protonox-Kivy Multiplatform Framework

See `docs/README.md` for full documentation on the framework layout, tooling, and templates.

## Repository layout
- `framework/`: Normalized Protonox toolchain components (kivy, python-for-android, pyjnius, cython, build-tools, kibit3, bkibit, windowser, protonox-devtools) with docs and templates.
- `templates/`: Production-ready app starters ready for Buildozer and Android 15+.
- `tools/`: Automation scripts for environment setup and builds.
- `examples/`: Placeholder for official sample apps.
- `docs/`: Guides for setup, troubleshooting, and integrations.
